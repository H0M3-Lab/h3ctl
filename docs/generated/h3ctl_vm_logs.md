## h3ctl vm logs

Print VM console output (last 50 lines by default)

```
h3ctl vm logs [flags]
```

### Options

```
  -f, --follow   stream output until interrupted
  -h, --help     help for logs
```

### Options inherited from parent commands

```
      --cluster string   target cluster (overrides active context)
      --output string    output format: text|json (default "text")
      --vm string        target VM (used with vm/config/talos/svc commands)
```
