# h3ctl

CLI for managing local [Talos](https://www.talos.dev/) Kubernetes clusters on macOS and Linux.

**Documentation:** [github.com/H0M3-Lab/h3ctl/wiki](https://github.com/H0M3-Lab/h3ctl/wiki)

## Install

```sh
go install github.com/h0m3-lab/h3ctl@latest
```

## Build from source

```sh
git clone https://github.com/H0M3-Lab/h3ctl.git
cd h3ctl
make build   # → bin/h3ctl
```
